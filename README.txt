Python Version: 3.6.3

The project is about using machine learning on wine dataset to help in customer choose a wine according to his/her taste/choice of combination of preferences. The result is Visualized using D3.js library to help customer understand the choices available as well as client to give suggestion to customer.

How to run:

Option 1: you can directly execute python file named "topWines.py" inside the Visual_Analytics folder using command "python topWines.py" or "<python 3.0+ version> topWines.py". Finally, open the URL displayed on the terminal.  
If you find any exception on command prompt/terminal then install that particular package using "pip install <package name>".

Option 2: Open the project in PyCharm.
